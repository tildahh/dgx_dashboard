nvidia-smi is failing to start

If you have Docker integration enabled, click Restart on the dashboard container below. Otherwise, use docker restart dashboard to restart.

If the logs show Failed to initialize NVML: Unknown Error, this is a known issue and can be worked around by configuring Docker to use cgroupfs as the cgroup driver for containers. See forums.developer.nvidia.com/t/failed-to-initialize... and docs.nvidia.com/.../troubleshooting.html for more info.
